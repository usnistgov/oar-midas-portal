Advanced Search Help

Use the Advanced Search to filter DMPs and DAPs with multiple criteria:

Text Search

Search across record names, primary contacts, and organization units. Use the main search bar or expand Advanced Search for more options.

Filter Options

  • Organization Unit: Filter by NIST organization (autocomplete available)
  • Owner/Contact: Filter by record owner or primary contact (autocomplete available)
  • Resource Type: Select DMP, DAP, or both
  • Process Status: Filter by Published, Edit, or Reviewed status
  • Has Publication: Show only records with associated paper publications

Date Filters

  • Exact Date: Records modified on a specific date
  • Before Date: Records modified before a date
  • After Date: Records modified after a date
  • Between Dates: Records modified within a date range

Managing Filters

  • Save Filter: Save current filter settings for reuse
  • Load Filter: Apply previously saved filter settings
  • Clear Filters: Remove all filters and show all records

Table Features

  • Column Visibility: Toggle columns on/off using checkboxes
  • Column Reordering: Drag column headers to reorder
  • Selection: Check boxes to select records for download
  • Export: Download selected records or export entire table
